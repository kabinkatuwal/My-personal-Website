Document Structure

<!DOCTYPE html>

Degines the document type and HTML version.
<!DOCTYPE html>

<html>

Root element of an HTML page.
<html lang="en">.../html>
Attributes: lang, xmlns

<head>

Contains meta information about the document.
<head>
  <title>Page Title</title>
</head>../html>

<body>

Contains the visible page content.
<body>
  <h1>My Pagee</h1>
</body>

Metadata

<meta>

Defines metadata about the HTML document.
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Free Web turorials" >
Attributes:charset, content, http-equiv, name

<title>

Defines the title of the document(shown in browser tab).
<title>MY Awesome Page</title>

<link>

Defines the relationship between a document and an external resource(most often used to link to style sheets).
<link rel="stylesheet" href="styles.css">
Attributes:href, rel, media, type

<style>

Defines style for a document.
<style>
  background-color:#f0f0f0;
</style>

<script>

Defines a client-side script(JavaScript).
<script src="script.js"></script>
<script>
  alert('Hello World!');
</script>
Attributes:src, async, defer, type

Text Content

<h1> to <h6>

Heading tags from most important(h1) to last important(h6).
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

<p>

Defines a paragraph.
<p>This is a paragraph of text.</p>

<br>

Inserts a single line break.
First line<br>Second line

<hr>

Defines a thematic change in the content (horizontal rule).
<p>First section</p>
<hr>
<p>Second section</p>

<pre>

Defines preformatted text (preserves spaces and line breaks).
<pre>
  This text
  will appear
  exactly as
  written here.
</pre>

<blockquote>

Defines a section that is quoted from another source.
<blockquote cite="http://example.com">
  This is a long quotation from another source.
</blockquote>
Attributes: cite

<cite>

Defines the title of a work.
<p>My favorite book is <cite>The Great Gatsby</cite>.</p>

<code>

Defines a piece of computer code.
<code>console.log('Hello');</code>

<sub> and <sup>

Defines subscript and superscript text.
H<sub>2</sub>O
E = mc<sup>2</sup>

Semantic Elements

<header>

Represents introductory content, typically a group of introductory or navigational aids.
<header>
  <h1>Website Title</h1>
  <p>Tagline</p>
</header>

<nav>

Defines a section of navigation links.
<nav>
  <a href="/html/">HTML</a>
  <a href="/css/">CSS</a>
</nav>

<main>

Specifies the main content of a document.
<main>
  <h1>Main Content</h1>
  <p>Article content...</p>
</main>

<article>

Defines independent, self-contained content.
<article>
  <h2>Article Title</h2>
  <p>Article content...</p>
</article>

<section>

Defines a section in a document.
<section>
  <h2>Section Title</h2>
  <p>Section content...</p>
</section>

<footer>

Defines a footer for a document or section.
<footer>
  <p>Copyright © 2025</p>
</footer>

Lists

<ul>

Defines an unordered (bulleted) list.
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

<ol>

Defines an ordered (numbered) list.
<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>
Attributes: reversed, start, type

<li>

Defines a list item.
<li>List item content</li>
Attributes: value (for ordered lists)

<dl>, <dt>, <dd>

Defines a description list (terms and definitions).
<dl>
  <dt>Term</dt>
  <dd>Definition</dd>
</dl>

Tables

<table>

Defines a table.
<table>
  <tr>
    <th>Header</th>
    <td>Data</td>
  </tr>
</table>

<tr>

Defines a table row.
<tr>
  <th>Header</th>
  <td>Data</td>
</tr>

<th>

Defines a table header cell.
<th scope="col">Name</th>
Attributes: colspan, rowspan, scope

<td>

Defines a table data cell.
<td colspan="2">Data spanning 2 columns</td>
Attributes: colspan, rowspan

<caption>

Defines a table caption.
<caption>Monthly Savings</caption>

<thead>, <tbody>, <tfoot>

Groups header, body, and footer content in a table.
<table>
  <thead>
    <tr><th>Header</th></tr>
  </thead>
  <tbody>
    <tr><td>Data</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td></tr>
  </tfoot>
</table>

Forms

<form>

Creates an HTML form for user input.
<form action="/submit" method="post">
  <!-- Form elements go here -->
</form>
Attributes: action, method, enctype, target

<input>

Creates an input field.
<input type="text" name="username" placeholder="Enter username">
<input type="password" name="pwd">
<input type="checkbox" name="subscribe" checked>
<input type="radio" name="gender" value="male">
<input type="file" name="photo">
<input type="submit" value="Submit">
Attributes: type, name, value, placeholder, required, disabled, readonly, checked, maxlength, min, max, step, pattern

<label>

Defines a label for an input element.
<label for="username">Username:</label>
<input type="text" id="username" name="username">
Attributes: for

<textarea>

Defines a multi-line text input control.
<textarea name="message" rows="4" cols="50">
Default text goes here.
</textarea>
Attributes: name, rows, cols, placeholder, required

<select> and <option>

Creates a drop-down list.
<select name="cars">
  <option value="volvo">Volvo</option>
  <option value="saab" selected>Saab</option>
</select>
Attributes: name, multiple, size (for select); value, selected (for option)

<button>

Defines a clickable button.
<button type="button">Click Me!</button>
<button type="submit">Submit</button>
Attributes: type, name, value, disabled

<fieldset> and <legend>

Groups related elements in a form with a caption.
<fieldset>
  <legend>Personal Info:</legend>
  <!-- Form elements -->
</fieldset>

Media

<img>

Embeds an image.
<img src="image.jpg" alt="Description" width="500" height="300">
Attributes: src, alt, width, height, loading, srcset, sizes

<audio>

Embeds sound content.
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>
Attributes: controls, autoplay, loop, muted, preload

<video>

Embeds video content.
<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>
Attributes: controls, autoplay, loop, muted, preload, poster, width, height

<source>

Specifies multiple media resources for media elements.
<source src="movie.ogg" type="video/ogg">
Attributes: src, type, media, srcset, sizes

Links

<a>

Defines a hyperlink.
<a href="https://example.com" target="_blank" rel="noopener">Visit Example</a>
Attributes: href, target, rel, download, hreflang, type

<link>

Defines the relationship between a document and an external resource.
<link rel="stylesheet" href="styles.css">
Attributes: href, rel, media, type, sizes

<nav>

Defines navigation links (see Semantic Elements section).
